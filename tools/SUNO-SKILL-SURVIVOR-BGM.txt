Réglages Suno recommandés
-------------------------
- Mode : Instrumental / Custom (sans paroles)
- Durée : 2:00 à 2:30 (puis boucle dans le jeu)
- Eviter : voix, paroles, breaks trop longs, fade-out final (préférer une fin qui boucle)
- Exporter : MP3 192 kbps ou plus

Installation dans le projet
---------------------------
1. Télécharger le MP3 depuis Suno
2. Renommer en skill-survivor-bgm.mp3
3. Copier dans assets/skill-survivor-bgm.mp3
4. git add assets/skill-survivor-bgm.mp3 && push (Cloudflare redéploie)
5. Tant que le fichier n’existe pas, une musique procédurale temporaire joue en jeu.

Test : lancer une run Skill Survivor, case « Musique » cochée dans le HUD.
