BUILDER — mise en ligne (2 minutes)
====================================

Site en production : https://build-sim.pages.dev/

Hébergement : Cloudflare Pages (repo GitHub → deploy auto sur main)
API : functions/api/[[path]].js + KV BUILDER_KV

OPTION A — Push GitHub (recommandé)
------------------------------------
1. Installe https://cli.github.com/ puis : gh auth login
2. Double-clic sur tools\publier-en-ligne.bat
3. Cloudflare redéploie en 1-2 min sur https://build-sim.pages.dev/

Réglages Cloudflare Pages (déjà configurés normalement) :
- Build command : npm run build
- Build output : dist
- Functions directory : functions
- KV binding : BUILDER_KV → namespace builder-global-data

OPTION B — Lien temporaire (PC allumé)
--------------------------------------
1. Lance tools\lancer-site.bat
2. Dans un autre terminal : npx localtunnel --port 8765
3. Partage l’URL https://....loca.lt affichée

Test après deploy :
  https://build-sim.pages.dev/api/community-builds
  https://build-sim.pages.dev/api/minigames/skill-survivor

Détails complets : tools/CLOUDFLARE-PAGES.txt
