## Résumé en une phrase

**Demandes de build communautaires** (publier une aide sans partager ton build actuel), rail **Dernières demandes de build**, et **airdrop KRAPS** — token fan communautaire **sans valeur réelle**, purement pour le fun, avec **leaderboard**, missions quotidiennes, roue et parrainage.

---

## Demande de build (Build request)

- **Nouveau mode** dans le panneau de publication : bascule **Publier un build** ↔ **Demande de build** (bouton swap à côté du titre).
- **Objectif** : demander de l’aide pour un **niveau / stage / abyss** sans exposer ni publier la composition actuelle du simulateur.
- **Champs demande** :
  - **Niveau à faire** (cible selon le mode Stage ou Abyss),
  - **Votre power actuel** (CP, menu par paliers),
  - **Personnage souhaité** (Nox, Vel, Zen, Shu) avec **aperçu visuel** du personnage choisi.
- **Publication** : enregistrement local + API `/api/build-requests` quand le serveur est actif ; message de confirmation après envoi.
- **Connexion requise** en ligne pour publier une demande (comme pour les builds communautaires).

---

## Derniers builds & dernières demandes

Sous le panneau de publication, deux rails côte à côte :

| Rail | Contenu |
|------|---------|
| **Derniers builds** | Les builds communautaires les plus récents (meta stage / progression, lien vers l’onglet Communauté). |
| **Dernières demandes de build** | Les demandes d’aide les plus récentes : personnage, mode, cible, power de l’auteur ; clic pour ouvrir le détail dans Communauté. |

États vides gérés avec messages dédiés (« Aucune demande pour le moment », rappel pour demander depuis le panneau).

## Airdrop KRAPS

### Important — à lire avant de participer

**KRAPS** est un **token fan communautaire**, créé pour le fun sur le builder :

- **Aucune valeur monétaire** — ce n’est pas une crypto, pas un investissement, pas un actif échangeable.
- **Pas d’affiliation officielle** avec Spekter Agency.
- Les points **KRAPS** servent uniquement au **classement** et à la **gamification** sur le site.
- Un message de **disclaimer** s’affiche à la première participation (bouton « Compris »).

*(Des récompenses optionnelles en token SPEK pourront éventuellement être évoquées séparément — distinctes du « fun score » KRAPS.)*

### Fonctionnalités

- **Solde KRAPS** visible dans l’en-tête (widget profil) + section dédiée **KRAPS** dans le builder.
- **Claim quotidien** : +1 KRAPS par jour (connexion).
- **Roue quotidienne** (HTML5 Canvas) :
  - 6 parts **visuellement égales** (+1, +2, +3, +5, +10, LOST),
  - tirage réel côté **serveur** selon les probabilités,
  - animation qui s’arrête sur le **bon segment**,
  - 1 tour par jour.
- **Parrainage** : bouton **Invite your friend** dans le profil — copie du lien d’invitation avec **popup de confirmation** ; +10 KRAPS par invitation validée (max. 3 invitations).
- **Leaderboard** : classement des joueurs par total KRAPS (rang, pseudo, avatar, score).
- **Mini-jeu** : annoncé, **bientôt disponible**.
- **i18n** : FR, EN, PT-BR, ID, VI pour toute la section KRAPS.

### Éligibilité

- Compte connecté (Discord / Telegram selon config serveur).
- **Discord ou Telegram** suffit pour participer (pas besoin des deux).
