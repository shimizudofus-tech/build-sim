## Communauté & builds

- **Chat live** sur la zone communauté : discussion à côté du builder, position réglée pour rester alignée avec les contrôles et le bloc marché / Getgems.
- **Publication** : champ nom de build retiré ; **puissance (CP) obligatoire** à la publication avec menu **par paliers 50K** ; suggestion de **noms de builds** déjà utilisés ; chargement d’un build depuis une suggestion.
- **Filtre communauté par puissance (CP)** optionnel ; **tous les personnages** visibles par défaut dans les filtres ; tri des builds tenant compte d’une **puissance plus basse** quand pertinent.
- **Profil auteur** : plus de champ auteur libre — **profil du compte connecté** ; correctifs **avatars** ; avatar **dans le bouton profil** ; photo de profil **clic pour changer** (à la place d’un bouton upload séparé).
- **Derniers builds** (rail sous publier) : affichage de la **meta stage** / progression selon le mode.
- **Libellés builder** : retour à une présentation **ligne actifs / ligne passifs** pour le build courant (cohérence avec l’habitude « comme avant » pour actif/passif).
